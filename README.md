# Transfer Learning Lab Assignment

## Description

This project implements transfer learning using ResNet50 for brain tumor classification.

## Dataset

Brain MRI Dataset:
https://www.kaggle.com/datasets/navoneel/brain-mri-images-for-brain-tumor-detection

## Steps

1. Data preprocessing and augmentation
2. Model training using ResNet50
3. Fine-tuning
4. Evaluation with metrics

## Results

Achieved ~90% accuracy.

## Tools Used

* TensorFlow
* Keras
* Google Colab
